# Delivery App

## Level: Middle

## Live URL
https://delivery-app-kappa-ivory.vercel.app

## Repository
https://github.com/stasyzar/delivery-app

## Tech Stack
- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express + TypeScript
- Database: MongoDB Atlas

## Implemented Features
### Base Level
- Shops page with product listing
- Add to cart functionality
- Shopping cart page with quantity management
- Order form with validation (name, email, phone, address)
- Order saved to database on submit

### Middle Level
- Responsive design (desktop, tablet, mobile)
- Product filtering by category (multi-select dropdown)
- Product sorting (price asc/desc, name A→Z)
- Shop filtering by rating range (2-3, 3-4, 4-5)

> Note: Backend is hosted on Render free tier. 
> First load may take up to 60 seconds to wake up the server.
